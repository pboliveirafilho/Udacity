Para verificar se a poltica de "mulheres e crianas primeiro" foi aplicada, classifica-se os passageiros por idade como segue:

Funo para categorizar uma idade. Foram considerados com crianas os passageiros com menor do que 11 anos.

In [17]:
def categ_Age(age):
    if age >= 11:
        return "Adult"
    else:
        return "Child"

Funo para categorizar um Dataframe de idades

In [18]:
def categ_Ages(ages):
    ages = pd.DataFrame(ages)
    return ages.applymap(categ_Age)

Conclusões¶

1. Passageiros de classe scio-econmica (Pclass) mais elevada tiveram maior taxa de sobrevivncia?

Sim. Passageiros da primeira classe (Pclass = 1) apresentaram a maior taxa de sobrevivncia.

2. Passageiros com prioridade em situaes de emergncia (mulheres e crianas) tieram maior taxa de sobrevivncia?

Sim. O gnero feninino apresentou maior taxa de sobrevivncia, assim como as crianas (passageiros com idade inferior a 11 anos)

3. Passageiros que embarcaram em diferentes portos apresentaram diferentes taxas de sobrevivncia?

Sim. Passageiros que embarcaram no porto de Cherbourg apresentaram maior taxa de sobrevivncia.
