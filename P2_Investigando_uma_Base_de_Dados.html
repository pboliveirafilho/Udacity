INTRODUÇÃO¶

O naufrgio do RMS Titanic um dos mais desastrosos e famosos da histria. Em 15 de abril de 1912, em sua viagem inaugural, o Titanic afundou aps colidir com um iceberg, matando 1502 dentre 2224 pessoas (passageiros e tripulao). Esta tragdia chocou a comunidade internacional e resultou em avanos na segurana para navios.Uma das principais razes para o grande nmero de vtimas fatais neste naufrgio foi o nmero insuficiente de barcos salva-vidas para passageiros e tripulao.

O presente trabalho apresenta uma anlise dos dados demogrficos e informaes referentes a 891 dos 2224 passageiros e tripulantes a bordo do Titanic. O objetivo deste trabalho no apresentar concluses definitivas sobre a sobrevivncia de passageiros ou tripulantes do Titanic, mas sim investigar o conjunto de dados com base em um brainstorming de perguntas a serem respondidas no decorrer das anlises.

def category_Age(age):
    if age >= 60:
        return "Elderly"
    elif age >= 19:
        return "Adult"
    elif age >= 12:
        return "Teenager"
    else:
        return "Child"

A funo category_Ages ser utilizada para categorizar todos as valores da varivel Age.

In [12]:
def category_Ages(ages):
    ages = pd.DataFrame(ages)
    return ages.applymap(category_Age)

CONCLUSÕES¶

1. Passageiros de classe sócio-econômica (Pclass) mais elevada apresentaram maior proporção de sobreviventes?¶

Sim. Passageiros ou tripulantes da primeira classe (Pclass = 1) apresentaram a maior taxa de sobrevivncia.

2. Houve diferença entre as proporções de sobreviventes para homens e mulheres?¶

Sim. Passageiros e tripulantes do sexo femenino apresentaram maior taxa de sobrevivncia.

3. Qual faixa de idade apresentou maior proporção de sobreviventes?¶

A categoria Child (idade inferior a 12 anos) apresentou a maior taxa de sobrevivncia.

4. Passageiros com poucos parentes (irmãos, cônjugues, pais e filhos) apresentaram maior proporção de sobreviventes?¶

No necessariamente. Passageiros ou tripulantes desacompanhados (Family = 0) apresentaram taxa de sobrevivncia intermediria. Passageiros com 3 parentes a bordo do Titanic apresnetaram a maior taxa de sobrevivncia.

5. Passageiros que embarcaram em diferentes portos apresentaram diferentes proporções de sobreviventes?¶

Sim. Passageiros e tripulantes que embarcaram no porto de Cherbourg (C) apresentaram a maior taxa de sobrevivncia.
